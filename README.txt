1. До шаблонів додана можливість розпізнавати чи померлий власник. Для цього необхідно в полі тип документа підтверджуючого особистість вибрати "Інші документи". Та в полі коментарі записати ПІП спадкоємця.

1.1. Якщо власник помер, тоді в графічних шаблонах ПІП власника земельної ділянки відображатися не буде.

2. У разі відсутності межових знаків в "відомостях про встановлені межові знаки" буде відображатись посилання на ст. 106 ЗКУ (критерії відповідно до яких межові знаки не встановлюються). Також дані налаштування діють і для кадастрового плану.

3. Вдосконалено відображення адреси зем. діл. Область, район та назва територальної громади відмінються.

4. Додано можливість друку суміжників в відомості про встановлені межові знаки. Для відображення суміжника в відомості, необхідно його фамілію та ініціали додати в блок "Представник суміжника".
